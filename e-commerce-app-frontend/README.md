# Fullstack E-commerce App

A simple fullstack e-commerce project built with **React (frontend)** and **Node.js + Express (backend)**.  

## Features
- Browse products  
- Add/remove items to cart  
- Update quantity  
- Place orders (with checkout form)  
- Fully responsive UI  



## Project Structure
ecommerce-app/
├── backend/ # Express.js API
├── frontend/ # React.js app
